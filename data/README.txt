** Dataset used in the study **

Each dataset contains the taxonomical tree used during the training process

| Filename                    | Content                                                           |
| --------------------------- | ----------------------------------------------------------------- |
| WPD_mixed_BBCH.tar.gz       | The unnanotated Weed Phenological Dataset images                  |
| WPD_separated_BBCH.tar.gz   | The phenological stage annotated Weed Phenolofical Dataset images |
| DeepWeeds_256x256.tar.gz    | A selection of images from the DeepWeeds dataset*                 |
| PlantSeedlings_64x64.tar.gz | A selection of images from the PlantSeedlings dataset**           |

Note: Images have been resized from their original size to either 512x512px (WPD), 256 x 256 px (DeepWeeds) and 64 x 64 px (PlantSeedlings).
* Olsen, Alex, Dmitry A. Konovalov, Bronson Philippa, Peter Ridd, Jake C. Wood, Jamie Johns, Wesley Banks et al. "DeepWeeds: A multiclass weed species image dataset for deep learning." Scientific reports 9, no. 1 (2019): 2058.
